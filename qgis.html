QGIS Lab Images

Neon Airports


The data used to create this image was simple airport locations and lines connecting them to one another. To create the neon effect, the flight paths were given an outer glow and an additional white line was added add to the visual to exacerbate the neon-ness.


Bathymetry


The data used to create this image was multiple bathymetry layers that fit together giving the appearance of a continuous plane. The final image was created by utilizing the draw effects option for each layer, making each layer appear as a discrete object overlaid onto each lower layer. Thus making the finished project look like a woodcutting.


Jemez Image 1


The datat used to create this image and the next was a DEM and a polygon layer. The image shows the polygon layer draped over the DEM hillshaded layer in a way that it does not wash out. It was created by using the multiply blending mode on the polygon layer and then for the blending mode for the township layer, selecting the overlay option.


Jemez Image 2


This image uses the same data as the previous image as well as the same methods for creation. Only this image also has labels added in a way that they do not cross lines or appear behind the QGIS logo, which is done by changing the label blocking items option. They are also rendered using the soft light blending mode.


Denmark Roads


The data used to create this image are layers for Denmark's roads, water bodies, and country boundaries. The final image was created by using the addition option for feature blending mode.


USA


The data used to create this image is solely a layer containing the U.S. states. To create the final image 3 additional layers were created from the first. Topological coloring was used to make it so that the states were colored so that no adjacent states had the same color, the colors generated were done so at random. The shapeburst effect was used to make the color gradually fade into the state itself. The duplicate layers made use of the multiply rendering option and finally a road map was added for context.
